## What's Changed

### New Features
- Added GIF quality modes and profile presets for more controlled video-to-GIF output.
- Improved GIF look tuning so the faithful and vibrant modes produce better default results.

### Improvements
- Interactive command menus now support `Esc` as the soft-exit key while preserving normal typed menu search behavior.

### Bug Fixes
- Preserved alpha transparency in the quality GIF pipeline instead of flattening transparent content during palette generation.
